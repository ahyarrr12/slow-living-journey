# 🚀 slow-living-journey

Repository ini berisi tutorial pembelajaran dasar Frontend Development yang mencakup HTML, CSS, JavaScript Vanilla ES6. Setiap materi memiliki target pembelajaran, key result, dan project sederhana yang dapat digunakan sebagai bukti progres pembelajaran.

---

# 📚 Learning

## 1️⃣ HTML Fundamentals

Pelajari dasar-dasar HTML sebagai fondasi dalam membangun struktur halaman web.

### Materi

* Dasar HTML
* Struktur Dokumen HTML
* Heading dan Paragraph
* Link dan Image
* List dan Table
* Form HTML
* Semantic HTML

### Target Akhir

* Memahami struktur halaman web.
* Mampu membuat halaman web statis.
* Mampu menggunakan elemen semantik HTML5.

📄 Dokumentasi:
| HTML Fundamentals | [Open](./html/html.md) |
---

## 2️⃣ CSS Fundamentals

Pelajari CSS untuk mempercantik tampilan website dan membuat layout yang responsif.

### Materi

* Dasar CSS
* Selector
* Color dan Typography
* Box Model
* Flexbox
* CSS Grid
* Responsive Design

### Target Akhir

* Memahami styling website.
* Mampu membuat layout modern.
* Mampu membuat website responsif.

📄 Dokumentasi:
| CSS Fundamentals | [Open](./css/css.md) |

---

## 3️⃣ JavaScript Vanilla ES6

Pelajari JavaScript modern untuk membuat website yang interaktif.

### Materi

* Variabel dan Tipe Data
* Operator
* Conditional Statement
* Looping
* Function
* Arrow Function
* Template Literal
* Array dan Object
* DOM Manipulation
* Event Listener

### Target Akhir

* Memahami sintaks JavaScript modern.
* Mampu memanipulasi DOM.
* Mampu membuat aplikasi web sederhana.

📄 Dokumentasi:
| JavaScript ES6 | [Open](./JS%20Vanilla%20ES6/JSES6.md) |
