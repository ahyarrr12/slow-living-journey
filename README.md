# OKR Pembelajaran Web Development dan Golang

## 🎯 Objective 1: Menguasai Dasar HTML

Mampu memahami dan menerapkan struktur dasar HTML untuk membangun halaman web yang semantik.

### Key Results

* [ ] Menyelesaikan materi dasar HTML hingga 100%.
* [ ] Memahami penggunaan Heading, Paragraph, Link, Image, List, Table, dan Form.
* [ ] Membuat minimal 3 halaman web statis menggunakan HTML.
* [ ] Menggunakan elemen semantik HTML5 (`header`, `nav`, `section`, `article`, `footer`).
* [ ] Mengunggah seluruh latihan HTML ke GitHub.

---

## 🎨 Objective 2: Menguasai Dasar CSS

Mampu mempercantik dan mengatur tata letak halaman web menggunakan CSS.

### Key Results

* [ ] Memahami Selector, Property, dan Value pada CSS.
* [ ] Menerapkan CSS pada minimal 3 proyek HTML.
* [ ] Menggunakan Flexbox untuk membuat layout responsif.
* [ ] Menggunakan CSS Grid untuk mengatur tata letak halaman.
* [ ] Membuat 1 website responsif yang dapat diakses pada desktop dan mobile.
* [ ] Mengunggah seluruh latihan CSS ke GitHub.

---

## ⚡ Objective 3: Menguasai JavaScript Vanilla ES6

Mampu membuat website yang interaktif menggunakan JavaScript modern.

### Key Results

* [ ] Memahami Variable (`let`, `const`) dan Data Type.
* [ ] Memahami Conditional Statement dan Looping.
* [ ] Menggunakan Function dan Arrow Function.
* [ ] Memahami Template Literal dan Destructuring.
* [ ] Membuat minimal 5 latihan DOM Manipulation.
* [ ] Membuat proyek sederhana seperti To-Do List atau Kalkulator.
* [ ] Melakukan commit ke GitHub minimal 1 kali setiap hari.

---

## 🚀 Objective 4: Menguasai Dasar Golang (Go)

Mampu memahami dasar pemrograman Golang dan membuat aplikasi sederhana.

### Key Results

* [ ] Memahami Variable, Data Type, dan Operator.
* [ ] Memahami Conditional Statement (`if`, `switch`).
* [ ] Memahami Looping (`for`).
* [ ] Membuat dan menggunakan Function.
* [ ] Memahami Array, Slice, dan Map.
* [ ] Memahami Struct dan Pointer.
* [ ] Membuat minimal 10 program latihan dasar Golang.
* [ ] Membuat aplikasi CLI (Command Line Interface) sederhana.
* [ ] Mengunggah seluruh source code ke GitHub.
