# 1) Memahami Dasar HTML

### 1.1 Memahami fungsi HTML dalam membangun struktur website

HTML (HyperText Markup Language) digunakan untuk membuat struktur dan konten halaman web.

### 1.2 Membuat file index.html

Buat file:

```text
index.html
```

### 1.3 Menjalankan file HTML

Buka file menggunakan browser untuk melihat hasilnya.

---

# 2) Memahami Struktur Dasar HTML

### 2.1 Struktur Dasar HTML

```html
<!DOCTYPE html>
<html>
<head>
    <title>Website Pertama</title>
</head>
<body>

</body>
</html>
```

---

# 3) Menguasai Elemen Teks HTML

### 3.1 Heading

```html
<h1>Judul Utama</h1>
<h2>Sub Judul</h2>
```

### 3.2 Paragraf

```html
<p>Ini adalah paragraf.</p>
```

### 3.3 Format Teks

```html
<strong>Tebal</strong>
<em>Miring</em>
<mark>Highlight</mark>
```

---

# 4) Menguasai Multimedia dan Navigasi

### Menampilkan Gambar

```html
<img src="foto.jpg" alt="Foto Profil">
```

### Membuat Link

```html
<a href="https://example.com">Kunjungi Website</a>
```

---

# 5) Menguasai List dan Tabel

### List

```html
<ul>
  <li>HTML</li>
  <li>CSS</li>
</ul>
```

### Tabel

```html
<table>
  <tr>
    <th>Nama</th>
    <th>NIM</th>
  </tr>
</table>
```

---

# 6) Menguasai Form HTML

### Form Dasar

```html
<form>
  <input type="text">
  <button>Kirim</button>
</form>
```
